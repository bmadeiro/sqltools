Edit menu commands

The Edit menu offers the following commands:

Undo Undo the last action
Redo Redo the previously undone action

Cut Cut the selection and put it on the Clipboard
Copy Copy the selection and put it on the Clipboard
Paste Insert Clipboard contents

Cut Append Cut the selection and append it on the Clipboard
Cut Bookmarked Lines Cut the bookmarked lines and put them on the Clipboard
Copy Append Copy the selection and append it on the Clipboard
Copy Bookmarked Lines Copy the bookmarked lines and put them on the Clipboard.
This command can be extremally useful in combination Find... -> Mark All -> This One.

Delete Delete the current character or selection
Delete to End of Word Delete the word to the right
Delete Word Back Delete the word to the left
Delete Line Delete the current line
Delete Bookmarked Lines Delete the bookmarked lines

Select Word Selet the word in the current position
Select Line Select the current line
Select All Select the entire document

Scroll Up Scroll the view content up
Scroll To Center Scroll the current line in the center of view
Scroll Down Scroll the view content down

File menu commands

The File menu offers the following commands:

New Creates a new document.
New... Creates a new document with category (Text/PL/SQL/C++/...) and name from defaults.
Open Opens an existing document.
Reload Discards all changes and loads data from the file.

Close Closes an opened document.
Close All Closes all opened document. It shows confirmation dialog for all modified documents.

Save Saves an opened document using the same file name.
Save As Saves an opened document to a specified file name.
Save All Saves all opened documents.

Show file Location Dislay the file in File Panel.
Properties Allows to change document category (Text/PL/SQL/C++/...), tab/indent settings and new line style (DOS/Unix/Mac). It also shows the file full path and memory consumption.

Page Setup... Allows to customize page header, footer and page margins.
Print Preview... Displays the document on the screen as it would appear printed.
Print Setup... Selects a printer and printer connection.
Print... Prints a document.

Send Mail Sends the file as an attachment with Windows Email client.

Exit Exits SQLTools.

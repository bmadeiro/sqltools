Search menu commands

The Search menu offers the following commands:

Find... Find the specified text
Replace... Replace specific text with different text
Find next Find the next occurrance of the specified text
Find previous Find the previous occurrance of the specified text

Go To... Go to the specified location. The location can be specified relatively the first line of file, current line or the end of file.

Toggle Bookmark Toggles a bookmark for the current line on and off
Next Bookmark Move to the line containting the next bookmark
Previous Bookmark Move to the line containting the previous bookmark
Remove all Bookmarks Remove all bookmarks in the window\nRemove all Bookmarks

Set Random Bookmark Set a random access bookmark (from 0 to 9)
Get Random Bookmark Move to a random access bookmark (from 0 to 9)

Find Match Find the matching brace/keyword
Find Match Select Find the matching brace/keyword and select the text between braces

Find In Files... Seaches for a string in multiple files. The current implementation calls grep utility and captures its output.

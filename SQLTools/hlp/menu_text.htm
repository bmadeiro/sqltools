Text menu commands

The Text menu offers the following commands:

Stream Selection Toggle Stream Mode selection
Column Selection Toggle Columnar Mode selection
Toggle Selection Mode Toggle selection mode

Sort Sort the selected text or the whole file

Indent Selection Indent the selected text
Undent Selection Outdent the selected text

Untabify Selection Replaces tabs with spaces in the selection
Tabify Selection (All) Replaces spaces with tabs in the selection
Tabify Selection (Leading) Replaces leading spaces with tabs in the selection"

Normalize Keyword Normalize the word or selected text
Lowercase Lowercase the word or selected text
Uppercase Uppercase the word or selected text
Capitalize Capitalize the word or selected text
Invert case Invert case the word or selected text

Comment Line Comment the selected lines
Uncomment Line Uncomment the selected lines

Expand Template Expand Template (defined in "Templates" settings)
Datetime Stamp Print datetime stamp (defined in "Editing" settings)
Open file under cursor Open a file which the path is under cursor in the current window, in a new window.

Visible Spaces Show or hide space/tab characters
Line Numbers Show or hide line numbers
Ruler Show or hide the ruler
Column Markers Show or hide Column Markers (defined in "Categories" settings)
Indent Guide Show or hide Indent Guide

View menu commands

The View menu offers the following commands:

File Toolbar Show or hide the toolbar
SQL Toolbar Show or hide the toolbar
Connection Toolbar Show or hide the toolbar

Object Viewer Show or hide "Object Viewer"
Objects List Show or hide "Objects List"
Find in Files Output Show or hide "Find in Files" output

File Panel Show or hide "File Panel"
Workbook Show or hide documents as worksheets in a workbook
Status Bar Show or hide the status bar

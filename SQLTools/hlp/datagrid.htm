The Query (result) tab displays a data grid for executed SQL statements and scripts that return data.

Tips for Working with Data
  • This data grid can be pivoted, which consists of swapping the rows with the columns to provide a different perspective of the same data;
  • An individual data cell can be open with Popup viewer, which can display text / html or xml content;
  • The data grid allows range selection and supports copy or drag&drop selected cells in multiple formats

Range Selection / Copy / Export

You can select a range by using either the keyboard or the mouse. An entire row or column can also be selected with mouse.

The selected range can be copied either in a default format (one of text, html, sql ...) or chosen the current operation only.
The last can be done with pressing <Shift> while invoking Copy from the context menu.
The same is applied to Export and Open In Editor commands.

Drag and Drop

Drag and Drop is an essential feature of the data grid. You can drag the context selected cells or column headers and drop into the editor.
<Shift> and <Ctrl> let you choose a desired format:

It is worth to mention that dragging the left top corner of the grid in the editor inserts the complete column list.

Popup Viewer

if data grid cell content is not completely visible because of size you might want to see in the Popup Viewer.
The Popup Viewer can be activated with <Alt> + Left mouse button click or Middle mouse button click.

Settings

Tips
  • Size of LOBs, CLOBs and BLOBs is limited by "Max column fetch size" of "Data Grid 1" settings
  • Display formats for dates, timestamps, numbers can be changed in "Data Grid 2" settings
  • You can drag and drop in the editor any data grid cell
  • Drag and drop of the top left cell gives the list of all data grid columns

Object Viewer

Object Viewer is better alternative to SQL*Plus DESC command. It is designed to work with the editor.
Press F12 to pass the current word (under the cursor) / selection into Object Viewer, if it is a valid database object
then it will be displayed in the viewer. You also can enter an object name into the top field of Object Viewer.

Drag and Drop

If mutltiple objects are selected for a drag and drop operation then default format is applied to every item
and additionally the result can be formatted in COLUMN with <Ctrl> pressed.
<Shift> is always ignored in the case of multiple selection.

Query and Count query

Object Viewer lets you make a quick (count) query from selected table (or partition/subpartition).
You also can select multiple columns belong that table (or partition/subpartition). If no columns are selected, * is used.
However only a single object can be chosen for "FROM" clause so it is either none, table, partition or subpartition.

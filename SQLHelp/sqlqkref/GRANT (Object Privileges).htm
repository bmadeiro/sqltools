GRANT (Object Privileges)

Use this command to grant privileges for a particular object to users and roles. To grant system privileges and roles, use the GRANT command (System Privileges and Roles). 

 GRANT
   { object_priv | ALL [PRIVILEGES] }
   [ ( column [, column] ...) ]
   [, { object_priv | ALL [PRIVILEGES] }
      [ ( column [, column] ...) ] ] ...
 ON [ schema.| DIRECTORY] object
 TO { user | role | PUBLIC} ...
    [ WITH GRANT OPTION]

See also: GRANT (System Privileges and Roles) , REVOKE (Schema Object Privileges)

CREATE TABLESPACE

Use this command to create a tablespace. A tablespace is an allocation of space in the database that can contain schema objects. 

 CREATE TABLESPACE tablespace
 DATAFILE filespec
   [ AUTOEXTEND
     { OFF | ON
       [ NEXT integer [K | M] ]
       [ MAXSIZE
         { UNLIMITED | integer [K | M] } ] } ]
   [ LOGGING | NOLOGGING]
   [, filespec
     [ AUTOEXTEND
       { OFF | ON
         [ NEXT integer [K | M] ]
         [ MAXSIZE
           { UNLIMITED | integer [K | M] } ] } ] ]
   [LOGGING | NOLOGGING] ...
   [ MINIMUM EXTENT integer [K | M] ]
   [ DEFAULT STORAGE storage_clause
     | {ONLINE | OFFLINE}
     | {PERMANENT | TEMPORARY} ] ...

See also: ALTER TABLESPACE , DROP TABLESPACE

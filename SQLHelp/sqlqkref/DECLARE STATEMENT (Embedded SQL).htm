DECLARE STATEMENT (Embedded SQL)

Use this command to declare an identifier for a SQL statement or PL/SQL block to be used in other embedded SQL statements. 

 EXEC SQL [ AT {db_name | :host_variable} ]
 DECLARE STATEMENT { statement_name
                   | block_name} STATEMENT

See also: DELETE , INSERT , SELECT , UPDATE

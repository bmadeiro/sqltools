TRUNCATE

Use this command to remove all rows from a table or cluster and reset the STORAGE parameters to the values when the table or cluster was created. 

 TRUNCATE
   { TABLE [ schema. ]table
     [ [ PRESERVE | PURGE ] SNAPSHOT LOG ]
     | CLUSTER [ schema. ]cluster }
   [ { DROP | REUSE } STORAGE ]

See also: DELETE , DROP TABLE

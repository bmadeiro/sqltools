ROLLBACK (Embedded SQL)

Use this command to end the current transaction, discard all changes in the current transaction, and release all locks and optionally release resources and disconnect from the database. 

 EXEC SQL
   [ AT {db_name | :host_variable} ]
   ROLLBACK [WORK]
   [ { TO [SAVEPOINT] savepoint [RELEASE] | PUBLIC } ]

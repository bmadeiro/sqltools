ANALYZE

Use this command to perform one of the following functions on an index or index partition, table or table partition, index-only table, or cluster:

  • to collect statistics about the schema object used by the optimizer and store them in the data dictionary
  • to delete statistics about the schema object from the data dictionary
  • to validate the structure of the schema object
  • to identify migrated and chained rows of the table or cluster
  • to collect statistics on scalar object attributes
  • to validate and update object references (REFs) 

 ANALYZE {INDEX | TABLE | CLUSTER}
         [schema.]{index [PARTITION (partition_name)]
                   | table [PARTITION (partition_name)]
                   | cluster}
   { COMPUTE STATISTICS [FOR for_clause]
   | ESTIMATE STATISTICS [FOR for_clause]
                         [SAMPLE integer {ROWS | PERCENT} ]
   | DELETE STATISTICS
   | VALIDATE REF UPDATE
   | VALIDATE STRUCTURE [CASCADE]
   | LIST CHAINED ROWS [INTO [schema.]table] }
 
 for_clause
   [ FOR TABLE
   | FOR ALL [INDEXED] COLUMNS [SIZE integer]
   | FOR COLUMNS [SIZE integer] column | attribute [SIZE integer]
                             [column | attribute [SIZE integer] ] ...
   | FOR ALL [LOCAL] INDEXES]

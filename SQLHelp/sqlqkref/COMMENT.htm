COMMENT

Use this command to add a comment about a table, view, snapshot, or column into the data dictionary. 

 COMMENT ON
   { TABLE [schema.]{table | view | snapshot}
   | COLUMN [schema.]{ table.column
                      | view.column
                      | snapshot.column} } IS 'text'

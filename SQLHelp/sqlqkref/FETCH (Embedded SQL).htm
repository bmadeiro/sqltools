FETCH (Embedded SQL)

Use this command to retrieve one or more rows returned by a query, assigning the select list values to host variables. 

 EXEC SQL [FOR :host_integer] FETCH cursor
   { INTO :host_variable [ [INDICATOR] :indicator_variable]
       [, :host_variable [ [INDICATOR] :indicator_variable] ] ...
   | USING DESCRIPTOR descriptor}

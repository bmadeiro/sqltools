CREATE DIRECTORY

Use CREATE DIRECTORY to create a schema object (directory), which represents an operating system directory for administering access to, and the use of, BFILEs stored outside the database. A directory is an alias for a directory name on the server's file system where the files are actually located. 

 CREATE [OR REPLACE] DIRECTORY directory AS 'path_name'

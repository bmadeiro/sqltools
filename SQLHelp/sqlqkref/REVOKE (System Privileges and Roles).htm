REVOKE (System Privileges and Roles)

Use this command to revoke system privileges and roles from users and roles. To revoke object privileges from users and roles, use the 

 REVOKE command (Object Privileges).
 
 REVOKE
   { system_priv | role}
   [, { system_priv | role} ] ...
 FROM
   { user | role | PUBLIC}
   [, {user | role | PUBLIC} ] ...

See also: GRANT (System Privileges and Roles) , REVOKE (Schema Object Privileges)

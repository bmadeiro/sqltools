UPDATE

Use this command to change existing values in a table or in a view's base table. 

 UPDATE
   { [ schema. ]
     { table
       [ PARTITION ( partition_name )
       | @dblink ]
     | [ view | snapshot ] }
   [ @dblink ]
   | [ THE ] ( subquery_1 )
   [ t_alias ] }
 SET
   { ( column [, column ] ... ) = ( subquery_2 )
   | column = { expr | ( subquery_3 ) } }
   [, { ( column [, column ] ... ) = ( subquery_2 )
      | column = { expr | ( subquery_3 ) } } ] ...
   [ WHERE condition ]

See also: DELETE , EXPLAIN PLAN , INSERT , SELECT

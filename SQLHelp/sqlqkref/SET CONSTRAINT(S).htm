SET CONSTRAINT(S)

Use SET CONSTRAINT(S) to set, per transaction, whether a deferrable constraint is checked following each DML statement (IMMEDIATE) or when the transaction is committed (DEFERRED). 

 SET CONSTRAINT[S]
   { constraint [, constraint ] ... | ALL }
   { IMMEDIATE | DEFERRED }

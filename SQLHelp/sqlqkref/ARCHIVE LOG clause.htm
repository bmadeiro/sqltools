ARCHIVE LOG clause

Use this command to manually archive redo log file groups or to enable or disable automatic archiving. 

 ARCHIVE LOG [THREAD integer]
   { { SEQUENCE integer
     | CHANGE integer
     | CURRENT
     | GROUP integer
     | LOGFILE 'filename'
     | NEXT
     | ALL
     | START} [TO 'location']
   | STOP}

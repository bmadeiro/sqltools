REVOKE (Schema Object Privileges)

Use this command to revoke object privileges for a particular object from users and roles. To revoke system privileges or roles, use the 

 REVOKE command (System Privileges and Roles).
 
 REVOKE
   { object_priv | ALL [PRIVILEGES] }
   [, {object_priv | ALL [PRIVILEGES] } ] ...
 ON
   [ schema.| DIRECTORY] object
 FROM
   { user | role | PUBLIC}
   [, {user | role | PUBLIC} ] ...
   [ CASCADE CONSTRAINTS]

See also: GRANT (Object Privileges) , REVOKE (System Privileges and Roles)

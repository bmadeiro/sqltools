INSERT

Use INSERT to add rows to:

  • a table
  • a view's base table
  • a partition of a partitioned table
  • an object table
  • an object view's base table 

 INSERT INTO
   { [ schema.]{ table [PARTITION (partition_name) | @dblink]
                | view}
                [ @dblink]
                | [THE] (subquery_1) }
   [ ( column [, column] ...)]
   { VALUES (expr [expr] ...)
   | subquery_2 }
   [ REF INTO data_item]

See also: DELETE , EXPLAIN PLAN , SELECT , UPDATE

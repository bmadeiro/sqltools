CREATE PACKAGE

Use this command to create the specification for a stored package. A package is an encapsulated collection of related procedures, functions, and other program objects stored together in the database. The specification declares these objects. 

 CREATE [OR REPLACE] PACKAGE [schema.]package
   {IS | AS} pl/sql_package_spec

See also: ALTER PACKAGE , DROP PACKAGE

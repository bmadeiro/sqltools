DECLARE CURSOR (Embedded SQL)

Use this command to declare a cursor, giving it a name and associating it with a SQL statement or a PL/SQL block. 

 EXEC SQL [ AT {db_name | :host_variable} ]
 DECLARE cursor CURSOR FOR { SELECT command
                           | statement_name
                           | blockname }

See also: SELECT

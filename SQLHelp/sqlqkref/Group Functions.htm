Group Functions

Return a single row based on groups of rows, rather than on single rows

Function

Value Returned

AVG([DISTINCT|ALL] n)

Average value of n, ignoring nulls

COUNT([ALL] *)

Number of rows returned by a query or subquery

COUNT([DISTINCT| ALL] expr)

Number of rows where expr is not null

MAX([DISTINCT|ALL] expr)

Maximum value of expr

MIN([DISTINCT|ALL] expr)

Minimum value of expr

STTDEV([DISTINCT|ALL] n)

Standard deviation of n, ignoring null values

SUM([DISTINCT|ALL] n)

Sum of values of n

VARIANCE ([DISTINCT|ALL] n)

Variance of n, ignoring null values

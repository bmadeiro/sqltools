EXPLAIN PLAN

Use this command to determine the execution plan Oracle follows to execute a specified SQL statement. This command inserts a row describing each step of the execution plan into a specified table. If you are using cost-based optimization, this command also determines the cost of executing the statement. 

 EXPLAIN PLAN [ SET STATEMENT_ID = 'text' ]
   [ INTO [schema.]table [@dblink] ] FOR statement

See also: DELETE , INSERT , SELECT , UPDATE

CREATE CLUSTER

Use this command to create a cluster. A cluster is a schema object that contains one or more tables that all have one or more columns in common. 

 CREATE CLUSTER [schema.]cluster (column datatype
        [, column datatype] ...)
   [ PCTUSED integer
   | PCTFREE integer
   | INITRANS integer
   | MAXTRANS integer
   | SIZE integer [K | M]
   | TABLESPACE tablespace
   | STORAGE storage_clause
   | {INDEX | HASHKEYS integer [HASH IS expr] } ] ...
   [PARALLEL parallel_clause]
   [CACHE | NOCACHE]

See also: ALTER CLUSTER , DROP CLUSTER

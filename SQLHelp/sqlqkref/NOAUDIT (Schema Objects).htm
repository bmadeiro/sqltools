NOAUDIT (Schema Objects)

Use this command to stop auditing chosen by the AUDIT command (Schema Objects). To stop auditing chosen by the AUDIT command (SQL Statements), use the NOAUDIT command (SQL Statements). 

 NOAUDIT object_opt [, object_opt] ...
   ON [schema.]object [WHENEVER [NOT] SUCCESSFUL]

This SQL Reference is not up-to-date. For detailed information on this command, see the Oracle8i Server SQL Reference.

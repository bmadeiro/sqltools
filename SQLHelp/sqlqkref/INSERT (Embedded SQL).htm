INSERT (Embedded SQL)

Use this command to add rows to a table or to a view's base table. 

 EXEC SQL
   [ AT { db_name | :host_variable } ]
   [ FOR :host_integer ]
 INSERT INTO
   { [schema.]
     { table | view }
     [ @dblink ]
     | ( subquery_1 ) }
   [ ( column [, column] ...)]
   { VALUES (expr [expr] ...)
   | subquery_2 }

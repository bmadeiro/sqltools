DELETE

Use this command to remove rows from a table, a partitioned table, a view's base table, or from a view's partitioned base table. 

 DELETE [FROM]
   { [schema.]{table [PARTITION (partition_name) | @dblink]
               | view [@dblink] }
   | [THE] ( subquery ) }
   [alias]
   [WHERE condition]

See also: EXPLAIN PLAN , INSERT , SELECT , TRUNCATE , UPDATE

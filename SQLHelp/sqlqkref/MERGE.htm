MERGE

The MERGE statement is designed to combine multiple operations to reduce the complexity of mixed insert and update operations. 

  MERGE [hint] INTO [schema.]table [t_alias]
    USING [[schema.]table | view | subquery] t_alias
      ON ( condition ) [merge_update_clause | merge_insert_clause]

    merge_update_clause:

    WHEN MATCHED THEN
      UPDATE SET [<column> = [<expr>|DEFAULT][,]]
        <where_clause>
      DELETE
        <where_clause>

    merge_insert_clause:

    WHEN NOT MATCHED THEN INSERT ( <column> [,])
      VALUES (<expr>|DEFAULT[,])
        <where_clause>

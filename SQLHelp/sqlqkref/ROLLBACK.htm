ROLLBACK

Use this command to undo work done in the current transaction.

You can also use this command to manually undo the work done by an in-doubt distributed transaction. 

 ROLLBACK [WORK] [TO [SAVEPOINT] savepoint | FORCE 'text']

See also: COMMIT , SAVEPOINT , SET TRANSACTION

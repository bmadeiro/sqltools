ALTER SNAPSHOT LOG

Use this command to change the storage characteristics of a snapshot log. 

 ALTER SNAPSHOT LOG ON [schema.]table
   [ ADD {[PRIMARY KEY] [,ROWID]
         [,(filter_column) | ,(filter_column)] ...}
   [ PCTFREE integer
   | PCTUSED integer
   | INITRANS integer
   | MAXTRANS integer
   | STORAGE storage_clause] ...

See also: CREATE SNAPSHOT LOG , DROP SNAPSHOT LOG , ALTER SNAPSHOT

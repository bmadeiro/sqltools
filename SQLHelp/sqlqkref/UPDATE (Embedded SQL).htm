UPDATE (Embedded SQL)

Use this command to change existing values in a table or in a view's base table. 

 EXEC SQL
   [ AT { dbname | :host_variable } ]
   [ FOR :host_integer ]
 UPDATE
   { [ schema. ]
     { table | view | snapshot }
     [ @dblink ]
     | ( subquery_1 ) }
   [ t_alias ]
 SET
   { ( column [, column ] ... ) = ( subquery_2 )
   | column = { expr | ( subquery_3 ) } }
   [, { ( column [, column ] ... ) = ( subquery_2 )
      | column = { expr | ( subquery_3 ) } } ] ...
   [ WHERE condition ]

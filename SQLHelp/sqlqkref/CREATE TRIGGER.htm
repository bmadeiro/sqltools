CREATE TRIGGER

Use this command to create and enable a database trigger. A database trigger is a stored PL/SQL block that is associated with a table. Oracle automatically executes a trigger when a specified SQL statement is issued against the table. 

 CREATE [OR REPLACE] TRIGGER [schema.]trigger
   {BEFORE | AFTER | INSTEAD OF}
   { DELETE
   | INSERT
   | UPDATE [OF column [, column] ...] }
   [ OR { DELETE
        | INSERT
        | UPDATE [OF column [, column] ... ] } ] ...
   ON [schema.]{table | view}
   [ [ REFERENCING { OLD [AS] old | NEW [AS] new} ...]
     FOR EACH {ROW | STATMENT} [WHEN (condition) ] ] pl/sql_block

See also: ALTER TRIGGER , DROP TRIGGER

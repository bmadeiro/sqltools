LOCK TABLE

Use this command to lock one or more tables in a specified mode. This lock manually overrides automatic locking and permits or denies access to a table or view by other users for the duration of your operation. 

 LOCK TABLE
   [ schema.]{table | view} [@dblink]
   [, [schema.]{table | view} [@dblink] ] ...
 IN lockmode MODE
   [ NOWAIT]

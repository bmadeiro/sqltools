EXECUTE (Embedded SQL)

Use this command to execute a DELETE, INSERT, or UPDATE statement or a PL/SQL block that has been previously prepared with an embedded SQL PREPARE statement. 

 EXEC SQL [FOR :host_integer] EXECUTE { statement_name
                                      | block_name }
   [ USING {:host_variable [ [INDICATOR] :indicator_variable]
           [, :host_variable [ [INDICATOR] :indicator_variable] ] ...
           | DESCRIPTOR descriptor} ]
 

 EXECUTE (Anonymous PL/SQL Blocks) (Embedded SQL)
 ------------------------------------------------
 
 Use this command to embed an anonymous PL/SQL block into an Oracle
 Precompiler program.
 
 EXEC SQL [AT {db_name | :host_variable} ]
   EXECUTE pl/sql_block END-EXEC
 

 EXECUTE IMMEDIATE (Embedded SQL)
 --------------------------------
 
 Use this command to prepare and execute a DELETE, INSERT, or UPDATE
 statement or a PL/SQL block containing no host variables.
 
 EXEC SQL [AT {db_name | :host_variable} ]
   EXECUTE IMMEDIATE {:host_string | 'text'}

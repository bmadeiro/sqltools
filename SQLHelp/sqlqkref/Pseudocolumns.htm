Pseudocolumns

Pseudocolumns are similar to table columns. You can query any of these pseudocolumns, but you cannot change their values with DML statements.

Column name

Value

sequence.CURRVAL

Current value of sequence for the current session (sequence.NEXTVAL must be referenced first.)

sequence.NEXTVAL

Next value of sequence for the current session

[table.]LEVEL

1 for a root node, 2 for a child of a root, and so on. Used in the SELECT statement during tree-structured queries.

[table.]ROWID

Value that uniquely identifies a single row among other rows in the table. ROWID values are of datatype ROWID, not NUMBER or CHAR.

ROWNUM

Position of a single row among other rows selected by a query. Oracle server selects rows in an arbitrary order and evaluates ROWNUM before sorting rows for an ORDER BY clause.

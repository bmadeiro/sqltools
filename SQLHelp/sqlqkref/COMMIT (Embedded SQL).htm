COMMIT (Embedded SQL)

Use this command to end your current transaction, making permanent all its changes to the database and optionally freeing all resources and disconnecting from Oracle. 

 EXEC SQL [AT {db_name | :host_variable} ]
 COMMIT [WORK] [{ [COMMENT 'text'] [RELEASE]
                | FORCE 'text' [, integer] }]

See also: COMMIT

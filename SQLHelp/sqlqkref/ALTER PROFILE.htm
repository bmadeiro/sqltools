ALTER PROFILE

Use this command to add, modify, or remove a resource limit or password management in a profile. 

 ALTER PROFILE profile LIMIT
   { { SESSIONS_PER_USER
     | CPU_PER_SESSION
     | CPU_PER_CALL
     | CONNECT_TIME
     | IDLE_TIME
     | LOGICAL_READS_PER_SESSION
     | LOGICAL_READS_PER_CALL
     | COMPOSITE_LIMIT}
        {integer | UNLIMITED | DEFAULT}
   | { PRIVATE_SGA { integer [K | M]
                   | UNLIMITED
                   | DEFAULT}
   | FAILED_LOGIN_ATTEMPTS
   | PASSWORD_LIFETIME
   | {PASSWORD_REUSE_TIME
     |PASSWORD_REUSE_MAX}
   | ACCOUNT_LOCK_TIME
   | PASSWORD_GRACE_TIME}
       {integer | UNLIMITED | DEFAULT}
   | PASSWORD_VERIFY_FUNCTION
       {function | NULL | DEFAULT} } ...

See also: CREATE PROFILE , DROP PROFILE

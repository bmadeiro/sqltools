STORAGE clause

Use this command to specify storage characteristics for tables, indexes, clusters, and rollback segments, and the default storage characteristics for tablespaces. 

 STORAGE
   ( { INITIAL integer [ K | M ]
     | NEXT integer [ K | M ]
     | MINEXTENTS integer
     | MAXEXTENTS { integer | UNLIMITED }
     | PCTINCREASE integer
     | FREELISTS integer
     | FREELIST GROUPS integer
     | OPTIMAL [ integer [ K | M ] | NULL ] } ... )

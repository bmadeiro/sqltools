Filespec

Use this command to either specify a file as a data file or specify a group of one or more files as a redo log file group. 

 filespec (Data Files)
   'filename' [SIZE integer [K | M]] [REUSE]
 
 filespec (Redo Log File Groups)
   { 'filename'
   | (filename [, filename] ...)}
   [SIZE integer [K | M]] [REUSE]

NOAUDIT (SQL Statements)

Use this command to stop auditing chosen by the AUDIT command (SQL Statements). To stop auditing chosen by the AUDIT command (Schema Objects), use the NOAUDIT command (Schema Objects). 

 NOAUDIT
   { statement_opt | system_priv}
   [, {statement_opt | system_priv} ] ...
   [ BY user [, user] ...]
   [ WITH GRANT OPTION [NOT] SUCCESSFUL]

This SQL Reference is not up-to-date. For detailed information on this command, see the Oracle8i Server SQL Reference.

PARALLEL clause

This clause can only be used in the following commands:

  • ALTER CLUSTER
  • ALTER DATABASE ... RECOVER
  • ALTER INDEX ... REBUILD
  • ALTER TABLE
  • CREATE CLUSTER
  • CREATE INDEX
  • CREATE TABLE 

 { NOPARALLEL | PARALLEL
   ( { DEGREE {integer | DEFAULT}
     | INSTANCES {integer | DEFAULT} } ...) }

COMMIT

Use this command to end your current transaction and make permanent all changes performed in the transaction. This command also erases all savepoints in the transaction and releases the transaction's locks.

You can also use this command to manually commit an in-doubt distributed transaction. 

 COMMIT [WORK] [COMMENT 'text' | FORCE 'text' [, integer] ]

See also: ROLLBACK , SAVEPOINT , SET TRANSACTION

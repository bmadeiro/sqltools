GRANT (System Privileges and Roles)

Use this command to grant system privileges and roles to users and roles. To grant object privileges, use the GRANT command (Object Privileges). 

 GRANT
   { system_priv | role}
   [, { system_priv | role} ] ...
 TO
   { user | role | PUBLIC}
   [, { user | role | PUBLIC} ] ...
   [ WITH ADMIN OPTION]

See also: GRANT (Object Privileges) , REVOKE (System Privileges and Roles)

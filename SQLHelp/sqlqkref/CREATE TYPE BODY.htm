CREATE TYPE BODY

Use the CREATE TYPE BODY command to define or implement the member methods defined in the object type specification. 

 CREATE [OR REPLACE] TYPE BODY [schema.]type_name
   IS
     MEMBER {procedure_declaration | function_declaration};
     [ MEMBER {procedure_declaration | function_declaration}; ] ...
     [{MAP | ORDER} MEMBER function_declaration;]
   END;

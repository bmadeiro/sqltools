CREATE ROLE

Use this command to create a role. A role is a set of privileges that can be granted to users or to other roles. 

 CREATE ROLE role [NOT IDENTIFIED | IDENTIFIED {BY password | 
   EXTERNALLY | GLOBALLY} ]

See also: ALTER ROLE , DROP ROLE , GRANT (Object Privileges) , GRANT (System Privileges and Roles) , SET ROLE

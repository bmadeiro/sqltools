AUDIT (Schema Objects)

Use this command to choose a specific schema object for auditing. To choose particular SQL commands for auditing, use the AUDIT command (SQL Statements). 

 AUDIT object_opt [, object_opt] ...
   ON { [schema. | DIRECTORY] object | DEFAULT }
   [BY {SESSION | ACCESS} ]
   [WHENEVER [NOT] SUCCESSFUL]

See also: AUDIT (SQL Statements) , NOAUDIT (SQL Statements)

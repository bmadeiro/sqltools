Single-Row Number Functions

Single-row number functions accept numeric input and return numeric values.

Function

Value Returned

ABS(n)

Absolute value of n

ACOS(n)

Arc cosine of n

ATAN(n)

Arc tangent of n

CEIL(n)

Smallest integer greater than or equal to n

COS(n)

Cosine of n in radians

COSH(n)

Hyperbolic cosine of n

EXP(n)

e raised to the n th power

FLOOR(n)

Largest integer equal to less than n

LN(n)

Natural logarithm of n, where n > 0

LOG(m,n)

Logarithm, base m, of n

MOD(m,n)

Remainder of m divided by n

POWER(m,n)

m raised to the n th power

ROUND(n[,m])

n rounded to m decimal places; m defaults to 0

SIGN(n)

if n<0, -1; if n=0, 0; if n>0, 1

SIN(n)

Sine of n in radians

SINH(n)

Hyperbolic sine of n

SQRT(n)

Square root of n; if n<0, NULL

TAN(n)

Tangent of n in radians

TANH(n)

Hyperbolic tangent of n

TRUNC(n[,m])

n truncated to m decimal places; m defaults to 0

DECLARE TABLE (Embedded SQL)

Use this command to define the structure of a table or view, including each column's datatype, default value, and NULL or NOT NULL specification for semantic checking by the Oracle Precompilers. 

 EXEC SQL DECLARE table TABLE
   ( column datatype [ DEFAULT expr [NULL | NOT NULL]
                     | NOT NULL [WITH DEFAULT] ]
   [, column datatype [ DEFAULT expr [NULL | NOT NULL]
                      | NOT NULL [WITH DEFAULT] ] ] ...)

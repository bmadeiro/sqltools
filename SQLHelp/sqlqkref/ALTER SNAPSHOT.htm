ALTER SNAPSHOT

Use this command to alter a snapshot in one of the following ways:

  • changing its storage characteristics
  • changing its automatic refresh mode and times 

 ALTER SNAPSHOT [schema.]snapshot
   [ physical_attributes_clause ] ...
   [USING {INDEX [physical_attributes_clause] ...
          | [DEFAULT] MASTER ROLLBACK SEGMENT [rollback_segment] }
   [REFRESH [FAST | COMPLETE | FORCE]
   [START WITH date] [NEXT date] [WITH PRIMARY KEY ]
 
 physical_attributes_clause
   [ PCTFREE integer
   | PCTUSED integer
   | INITRANS integer
   | MAXTRANS integer
   | STORAGE storage_clause ]

See also: CREATE SNAPSHOT , DROP SNAPSHOT , ALTER SNAPSHOT LOG

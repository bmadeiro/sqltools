OPEN (Embedded SQL)

Use this command to open a cursor, evaluating the associated query and substituting the host variable names supplied by the USING clause into the WHERE clause of the query. 

 EXEC SQL OPEN cursor
   [ USING
     { :host_variable [ [INDICATOR] :indicator_variable] }
     [, { :host_variable [ [INDICATOR] :indicator_variable] } ] ...
     | DESCRIPTOR descriptor} ]

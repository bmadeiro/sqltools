SET ROLE

Use this command to enable and disable roles for your current session. 

 SET ROLE
   { role
     [ IDENTIFIED BY password ]
     [, role [ IDENTIFIED BY password ] ] ...
     | ALL [ EXCEPT role [, role ] ... ]
     | NONE }

See also: ALTER USER , CREATE ROLE
